Aplikācija

Lai sāktu darbu ar C++, mums vispirms nepieciešams izvēlēties aplikāciju, kurā veidot datorprogrammas.

Šī pulciņa laikā es jums iesaku izmantot Godbolt. Esmu izveidojis īsu video, kas parāda, kā sagatavot šo mājaslapu mūsu pulciņam.

Taču Godbolt nedrīkst izmantot Latvijas informātikas olimpiādēs, jo tā ir mājaslapa internetā, nevis datorprogramma. Tā vietā olimpiādēs atļauts izmantot Code::Blocks. Ja ir interese šo programmu izmantot jau uzreiz, tad Discord kanālā #ieteikumi var atrast instrukcijas.

Visas programmas šajā pulciņā sākas ar šādu šablonu: 

#include <bits/stdc++.h>

using namespace std;

typedef long long ll;
typedef long double ld;

int main(){
    ios_base::sync_with_stdio(0);
    cin.tie(NULL);
    // kods šeit

Ielīmējiet šo šablonu Godbolt katru reizi, kad sākat veidot jaunu programmu.

Mainīgo izvadīšana

Tāpat kā matemātikā, arī programmēšanā mēs daudz izmantojam mainīgos. Šonedēļ iepazīsimies ar \(ll\) (ell-ell) veida mainīgajiem.

\(ll\) ir mainīgais, kura vērtība ir vesels skaitlis. Piemēram, ja mēs vēlamies, lai \(x\) vērtība būtu \(53\), mēs varam uzrakstīt rindiņu:

ll x = 53;

Tagad, lai parādītu lietotājam skaitli \(53\), atliek uzrakstīt:

cout << x << endl;

Frāze \(cout\) nozīmē "character output" jeb "simbolu izvade", bet \(endl\) nozīmē "end line" jeb "rindas beigas".

Pamēģini ielīmēt zemāk doto programmu Godbolt! Cerams, ka labajā pusē zem frāzes Program stdout parādīsies skaitlis \(53\). To, kas parādās šajā labajā pusē, mēs sauksim par izvaddatiem.

#include <bits/stdc++.h>

using namespace std;

typedef long long ll;
typedef long double ld;

int main(){
    ios_base::sync_with_stdio(0);
    cin.tie(NULL);
    // kods šeit
    ll x = 53;
    cout << x << endl;
}

53

Mēs varam arī veikt vienkāršas operācijas ar skaitļiem.
